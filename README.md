Welcome to VineScript, the nostalgic script that takes your programs back to the 2012 era. Each expression elicits memories of simpler times.

## Features
* If/Else 
* While Loops
* True/False
* And/Not
* Arrays
* Statically Typed
* `.vine` file extension
* Nostalgia
